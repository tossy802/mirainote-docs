# Google Play ストア掲載情報

## アプリ名
ミライノート - ライフプラン × FIREシミュレーター

## 短い説明（80文字以内）
あなたのFIREまで、あと何日？ 4指数連動で目標達成日を毎日カウントダウン。

## 詳細な説明（4000文字以内）

【ミライノートとは？】
将来のお金を「見える化」する、ライフプラン × FIREシミュレーターアプリ。
あなたの月支出と保有資産から、経済的自由を達成するまでの日数を自動計算します。
複雑な金融計算はアプリにおまかせ。今日のあなたから、未来の自分までを、グラフ1枚でつなぎます。


🔥【新機能】FIREカウントダウン

「FIREって、結局いつ達成できるの？」
そんな曖昧な目標を、毎日見える「あと◯◯日」に変えます。

■ あなたのFIREまでの日数を、毎日表示
月支出と保有資産から、4%ルール（年間支出×25倍）など5つのFIREタイプ別に
目標額を自動計算。複利成長と月積立を加味して、達成までの日数を割り出します。

■ 4つの実在指数で、現実的な複利成長
- S&P 500（米国主要500社）
- オルカン（全世界株式）
- NASDAQ 100（米国テック大型）
- FANG+（FAANG含む大型テック10社）

過去の長期平均リターンで計算するので、「適当な5%」ではない説得力のある予測。
保有する指数の組み合わせに応じて、加重平均年率も自動算出します。

■ 資産推移グラフで、達成日を一目で
初期投資・累積積立・現金を色分けした積み上げグラフで、いつ何が起きるか視覚化。
5年後・10年後・15年後・達成時の総資産がマイルストーンで一覧表示されます。

■ 5つのFIREタイプから選べる
・リーンFIRE（4%ルール）
・ファットFIRE（3%ルール）
・サイドFIRE（副業前提）
・バリスタFIRE（パート前提）
・コーストFIRE（積立なしで65歳到達）

各タイプの根拠も解説付き。あなたのライフスタイルに合うゴールが選べます。

■ ライフプランからワンタップ取り込み
すでに作成済みのライフプランから、年齢・月支出・貯蓄をそのまま自動入力。
二度入力する手間はゼロです。


📊【ライフプラン機能】将来の家計をシミュレーション

■ 16カテゴリの細やかな入力
プロフィール / 収入 / 生活費 / お子さま / お住まい / その他収支 / 貯蓄・運用 /
積立投資 / 負債・保険 / 年金 / 老後 / 介護・医療 / 社会保障・税 /
リスク分析 / 相続・贈与 / 目標設定

■ 生涯キャッシュフロー表
年ごとの収入・支出・資産残高を一覧で確認。
将来の資産推移が一目でわかります。

■ ライフプランスコア
あなたのライフプランを100点満点で評価。改善ポイントもアドバイスします。

■ 複数プラン管理
「現状維持」「住宅購入後」など、複数のシナリオを作成して比較できます。


💎【こんな方におすすめ】

・FIREに興味があるが、「いつ達成できるか」が見えなくて不安な方
・将来のお金が足りるか不安な方
・住宅購入や教育費の計画を立てたい方
・老後の生活設計を考えたい方
・資産形成の目標を立てたい方
・自分でファイナンシャルプランを作りたい方


🔒【プライバシーへの徹底配慮】

・入力したデータは全てあなたの端末内にのみ保存
・指数価格データの取得を除き、外部サーバーへ個人情報を送信しません
・アカウント登録不要・無料でご利用いただけます
・完全オフラインでも（指数データは前回キャッシュで）動作


【注意事項】
・本アプリの計算結果は参考値です
・指数の長期想定年率は過去実績に基づく試算であり、将来を保証するものではありません
・実際の資産運用や人生設計は、ファイナンシャルプランナー等の専門家にご相談ください
・指数価格データはCloudflare Workers経由で日次配信されます


【対応端末】
Android 7.0以上
